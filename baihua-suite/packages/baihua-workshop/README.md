# Baihua Workshop

> 白桦工坊 — Sukačev 视频平台的后端服务。

## 这是什么

Baihua Workshop 是 [Sukačev 视频平台](http://suk-baihua.top) 的后端服务层，负责视频内容管理、用户系统、以及数据持久化。

## 关联产品

- [Sukačev 视频平台](http://suk-baihua.top) — 前端视频系统
- [SUK_白桦](https://space.bilibili.com/397661462) / [SUK_桦树工坊](https://space.bilibili.com/3632306778540781) — 内容源

## 技术栈
