# Fracture Workshop Alliance · 递砖者联盟

高密度认知者的生存基建与协作协议库。当主流系统无法容纳高密度认知者时，这些协议提供替代性的协作基础设施。

## 核心文档

| 文档 | 版本 | 说明 |
|------|------|------|
| NBP协议 | v0.1 | 神经-义体通信协议（Neural-Body Protocol） |
| KMP协议 | v0.1 | 密钥管理协议（Key Management Protocol） |
| 意识连续性评估框架 | v0.1 | 容器匹配与意识迁移评估标准 |
| 递砖者公约 | v0.1 | 联盟成员行为准则与协作规范 |

## 定位

这不是一个传统意义上的技术项目，而是一套为"裂缝中的建造者"设计的生存协议体系。递砖者联盟（Fracture Workshop Alliance）为那些处于主流系统边缘的高密度认知者提供：

- **通信协议** — 安全的神经-义体信息交换标准
- **身份管理** — 跨容器的意识连续性验证
- **协作规范** — 去中心化的贡献与治理框架
